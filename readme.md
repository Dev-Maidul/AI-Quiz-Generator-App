# 🧠 AI Quiz Generator App

An intelligent **AI-powered learning assistant** that converts images and audio into **structured notes, transcripts, and interactive quizzes**.

🚀 Built to simulate a real-world educational assistant using **Streamlit + Gemini API**

---

## 🔥 Live Demo

👉 https://ai-quiz-generator-app.streamlit.app/

---

## 📌 Key Features

* 📸 Upload images (notes, slides, handwritten content)
* 🧠 AI extracts and generates **clean structured notes**
* 🎧 Convert audio into **accurate transcripts**
* ❓ Automatically generate **quizzes from content**
* 💡 Helps reinforce learning through practice
* ⚡ Real-time processing with smooth UI
* 🎯 Clean and minimal interface (Streamlit)

---

## 🛠️ Tech Stack

* **Frontend/UI:** Streamlit
* **AI Integration:** Gemini API (Google Generative AI)
* **Image Processing:** PIL (Python Imaging Library)
* **Audio Processing:** gTTS
* **Environment Management:** Python venv + dotenv

---

## 🧠 How It Works

1. Upload an image or audio file  
2. AI processes the input  
3. Generates:
   * 📄 Structured Notes  
   * 🎧 Transcript (for audio)  
   * ❓ Quiz questions  
4. Displays results instantly in the UI

## ⚡ Installation & Setup

```bash
# Clone the repository
git clone https://github.com/Dev-Maidul/AI-Quiz-Generator-App.git

# Navigate to project
cd AI-Quiz-Generator-App

# Create virtual environment
python3 -m venv quizenv
source quizenv/bin/activate   # Mac/Linux

# Install dependencies
pip install -r requirements.txt
```

---

## 🔑 Environment Variables

Create a `.env` file in the root directory:

```env
GEMINI_API_KEY=your_api_key_here
```

---

## ▶️ Run the App

```bash
streamlit run app.py
```
